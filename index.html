How it works

Invisible at the OS level,
not a trick

No overlays. No display filters. No hacks. A single public, documented AppKit API call makes the window disappear from every capture pipeline on macOS.

01

NSWindow.sharingType = .none

This public AppKit API tells the macOS display server compositor to exclude this window from all capture operations — before any recording app, screenshot tool, or API can observe it.

02

Physical display only

The window renders normally on your monitor. It simply does not exist to capture pipelines — CGWindowListCreateImage, SCStreamConfiguration, AVCaptureScreenInput all return nothing.

03

Every tool is blind to it

Zoom, Google Meet, Microsoft Teams, OBS Studio, QuickTime, Cmd+Shift+5. None of them can capture what the compositor has excluded.

04

No Dock icon

Veil lives only in the menu bar as ⬡. No Dock presence, no window in Mission Control, no trace in Alt+Tab. Completely discreet.

Install

Up and running
in 30 seconds

$ open Veil-v1.0.1.dmg # drag to Applications

# Option B — Build from source
$ xcode-select --install
$ git clone https://github.com/rbc33/Veil.git && cd Veil
$ bash build.sh && open Veil.app

# ⬡ appears in your menu bar. Click → Open chat.
