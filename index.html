01 / HOW IT WORKS

ONE API
CALL.

// The entire secret. // One native AppKit API. window.sharingType = .none // Removes the window from // macOS display compositor // capture pipeline. // Before Zoom, OBS, or // CGWindowListCreateImage // ever sees it.
01

macOS compositor exclusion

NSWindow.sharingType = .none tells macOS to remove the window from the display server's capture pipeline at the compositor level — before any app can observe it.

02

Physical display only

The window renders on your monitor normally. It simply does not exist to CGWindowListCreateImage, SCStreamConfiguration, or any screen capture API.

03

Every recorder is blind

Zoom, Google Meet, Teams, OBS, QuickTime, Cmd+Shift+5. All confirmed invisible. Documented public API — not a hack.

04

No Dock icon

Lives only as ⬡ in the menu bar. No Mission Control entry, no Alt+Tab presence. Completely discreet.
